3. Implementacja InMemoryRoomRepository
👉 Tak – powinno implementować IRoomRepository (nie „być oddzielne”).
Dzięki temu możesz później podmienić np. na bazę danych bez zmiany reszty kodu.

public class InMemoryRoomRepository : IRoomRepository
{
    private readonly List<Room> _rooms = new List<Room>
    {
        new Room { Name = "Sala A", Capacity = 10 },
        new Room { Name = "Sala B", Capacity = 20 },
        new Room { Name = "Sala C", Capacity = 30 }
    };

    public IEnumerable<Room> GetAll()
    {
        return _rooms;
    }

    public Room GetByName(string name)
    {
        return _rooms.FirstOrDefault(r => r.Name == name);
    }

    public void Add(Room room)
    {
        _rooms.Add(room);
    }
}

🔹 Gdzie używać ObservableCollection?
👉 W ViewModelu:

public class RoomViewModel
{
    private readonly IRoomRepository _repository;

    public ObservableCollection<Room> Rooms { get; set; }

    public RoomViewModel(IRoomRepository repository)
    {
        _repository = repository;
        Rooms = new ObservableCollection<Room>(_repository.GetAll());
    }
}

✔ repo pozostaje czyste

🔹 Lepszy kierunek (jeśli będzie więcej rabatów)
Jeśli będziesz mieć kilka rabatów, możesz zrobić:

public class DiscountService
{
    private readonly IEnumerable<IDiscountPolicy> _policies;

    public DiscountService(IEnumerable<IDiscountPolicy> policies)
    {
        _policies = policies;
    }

    public decimal ApplyDiscounts(decimal price, int hours)
    {
        foreach (var policy in _policies)
        {
            price = policy.ApplyDiscount(price, hours);
        }

        return price;
    }
}
👉 wtedy każda polityka „próbuje” się zastosować

Zakładam, że masz w ViewModelu:

🧩 XAML (MainWindow.xaml)
<Window x:Class="RoomReservation.MainWindow"
        xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        Title="Rezerwacja sal" Height="400" Width="400">

    <StackPanel Margin="20">

        <!-- Lista sal -->
        <TextBlock Text="Wybierz salę:" FontWeight="Bold"/>
        <ListBox ItemsSource="{Binding Rooms}"
                 SelectedItem="{Binding SelectedRoom}"
                 DisplayMemberPath="Name"
                 Height="100"/>

        <!-- Liczba godzin -->
        <TextBlock Text="Liczba godzin:" Margin="0,10,0,0"/>
        <TextBox Text="{Binding Hours, UpdateSourceTrigger=PropertyChanged}" />

        <!-- Przycisk -->
        <Button Content="Rezerwuj"
                Command="{Binding ReserveCommand}"
                Margin="0,10,0,0"
                Height="30"/>

        <!-- Cena -->
        <TextBlock Text="Cena:" Margin="0,10,0,0" FontWeight="Bold"/>
        <TextBlock Text="{Binding Price}" FontSize="16"/>

    </StackPanel>
</Window>
🔹 Co tu się dzieje
ItemsSource → lista sal z repo

SelectedItem → wybrana sala

TextBox → liczba godzin

Button → wywołuje komendę

Price → wynik po rabacie
